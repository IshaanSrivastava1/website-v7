Real-Time Equity Charting App

Python · FastAPI · WebSockets · Alpaca API · TradingView Lightweight Charts · Render

View live app →

What I Built

  • Search any US stock and pull up an interactive candlestick chart on daily or weekly timeframes.
  • Real-time price streaming — the current candle and price update live from a market-data feed during trading hours.
  • Automatic gap detection plus a full set of technical indicators: EMA and SMA overlays, Bollinger Bands, and RSI & MACD sub-charts.
  • Click-and-drag trendline drawing — draw, move, and delete lines directly on the chart, saved per ticker so they persist between visits.

How It Works

A FastAPI backend serves both the JSON API and the charting frontend. Price history and all indicators are computed server-side with pandas/NumPy, and the charts are rendered with TradingView's Lightweight Charts library.

The tricky part was live data: the market feed's free tier allows only one streaming connection, so the server keeps a single shared connection and fans each incoming trade out to every visitor watching that ticker — meaning many people can use the site at once without exhausting the limit. The whole thing is deployed on Render behind one shareable URL.

Good to Know

Live prices stream during US market hours (9:30am–4:00pm ET, weekdays); outside those hours you'll see the latest historical candles, indicators, and drawing tools. The app is on a free host that sleeps when idle, so the first load may take ~30–60 seconds to wake up.
